# 20-supermall
  - 这是一个购物街项目，主要参考应用是蘑菇街，模仿coderwhy老师
  + 基于Vue全家桶+BetterScroll开发的移动端项目
  - 包括主页、详情页、购物车和我的四个部分

  + 为避免服务器接口暴露，requset.js文件中对服务器地址进行了修改
